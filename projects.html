Fabrication and Performance analysis of InGaN/GaN based Blue Light Emitting Diode
This project is a part of "LED fabrication process and performance evaluation" course (Summer 2020). Here, we have fabricated InGaN/GaN Multi Quantum Well (MQW) based blue Light Emitting Diodes (LED) in Clean room. The LED is fabricated in chip level following all the fabrication steps from LED structure growth via MOCVD to metal Pad deposition. The Electrical and Electroluminescence properties are analyzed. The fabricated LEDs have maximum light output power of 0.3 a.u. (arbitrary unit). All three fabricated samples can be used at 300 mA current. Full Report
Switching Characteristics of field effect transistor with different input parameters
This project was a part of the "Theory of Semiconductor Devices" course (Spring 2020). The devices that were simulated are: Enhancement type n-MOSFET, Enhancement type p-MOSFET, N-Channel JFE. Step by step n+/n- or p+/p- channel formation in a p/n type substrate was analyzed with Microwind. OrCAD Capture CIS was used to perform dc and ac small signal analysis of these devices. Also, gate voltage, capacitance between gate and drain etc. were varied and plotted.

Undergraduate Projects

Car Parking Using Microcontroller
This project was a part of the "Microcontroller & Robotics" short course. It was a miniature car parking model that indicated which parking lot was empty and which one was full. The project was completed using PIC 16F877A microcontroller with IR Sensor, display, led, etc. Proteus and mickroC were respectively utilized to design the full diagram and to do the coding works.
RFID based attendance system
An attendance system was designed using Radio-frequency identification (RFID). Whenever a person punched his card (with RFID tag), it recorded the attendance of the person. The system showed the total number of students present on a particular day (24 hours) and after the 24 hours time window, the memory of the previous day was cleared.
