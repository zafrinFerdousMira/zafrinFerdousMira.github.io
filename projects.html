Car Parking Using Microcontroller
This project was a part of the "Microcontroller & Robotics" short course. It was a miniature car parking model that indicated which parking lot was empty and which one was full. The project was completed using PIC 16F877A microcontroller with IR Sensor, display, led, etc. Proteus and mickroC were respectively utilized to design the full diagram and to do the coding works.
RFID based attendance system
An attendance system was designed using Radio-frequency identification (RFID). Whenever a person punched his card (with RFID tag), it recorded the attendance of the person. The system showed the total number of students present on a particular day (24 hours) and after the 24 hours time window, the memory of the previous day was cleared.
